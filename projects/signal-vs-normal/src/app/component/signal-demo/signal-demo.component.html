✅ Signal
OnPush

Signals are reactive by design. Angular tracks every read and schedules a precise re-render on every .set() or .update().

component.ts 
counter = signal(0);  // ← reactive!
name = signal('Vivek');
doubled = computed(() => this.counter() * 2);

increment() {
  // Angular KNOWS about this change 👇
  this.counter.update(v => v + 1);
}
